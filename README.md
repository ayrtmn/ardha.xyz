# ardha.xyz

Personal website of Ardha — a Master's student at TUM and government officer focusing on climate policy, sustainability, and data-driven analysis.

## Tech Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS v4
- MDX via next-mdx-remote
- Inter font
- Vercel Analytics & Speed Insights

## Features

- Minimalist, responsive design
- Dark/light theme toggle
- MDX-based blog posts
- SEO optimized (sitemap, RSS, JSON-LD)
- Dynamic OG images

## Content

All content is stored in the `content/` directory and can be edited via Markdown/MDX:

- `content/bio.md` — Homepage bio
- `content/about.mdx` — About page
- `content/now.mdx` — Now page
- `app/blog/posts/` — Blog posts

## Development

pnpm dev
